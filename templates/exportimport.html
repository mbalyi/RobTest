Export possibilities:

  • Export to .docx Microsoft Office Word format:

    • The exported Word document won't include html style formattings and images.

    One default template is available and nothing else.

  • Export to .pdf Portable Document Format format:

    • The exported PDF will include all html style formattings and images.

    It's templatable. You can upload a formatted html file to use it to the export formatting.

    You can upload and control the templates in the Templates menu.

  • Export to .xlsx Microsoft Office Excel format:

    • The exported Excel table sheet won't include html style formattings and images.

    In the first instance it's for result exporting and chart exporting.

    User's Guide:

    1. Select a case or set.
    2. If you want to export it to PDF you can select a html template to format the document. Otherwise skipp this step.
    3. Click on the selected export button.
    4. Server generates a download link and when it finished download link will be enabled.
    5. If you click on it the download will start.

    Export will start in a few second. Don't go to an other menu before the download started!
